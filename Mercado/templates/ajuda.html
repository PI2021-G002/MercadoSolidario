Mercado Solidário

Tutoriais e explicações

  • Para poder utilizar um produto por código de barras, o código desse produto real, deve estar associado a um código de produto solidário.
  • Caso haja novo produto para ser distribuído no Mercado Solidário CS, o mesmo deve ser cadastrado como produto solidário e pelo menos um código de barras de um produto real deve ser associado a ele. Um administrador pode fazer o cadastro do Produtto Solidário.
  • Em caso de dúvidas, contatar Beto ou Vitor.
